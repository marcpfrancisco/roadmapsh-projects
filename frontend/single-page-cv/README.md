# Single Page CV — roadmap.sh Project

This project is a solution to the [Single Page CV Project](https://roadmap.sh/projects/single-page-cv) from [roadmap.sh](https://roadmap.sh).  
It focuses on creating a **minimal, accessible, and responsive one-page CV** using only **HTML and CSS** — no frameworks or JavaScript.

---

## Project Overview

The goal of this project is to design and build a **single-page CV (Curriculum Vitae)** that displays key professional information such as:

- Full Name
- Contact Information
- Summary / Objective
- Skills
- Education
- Work Experience

This challenge helps developers strengthen their understanding of **semantic HTML**, **CSS layout**, and **responsive web design** principles.

---

## Learning Objectives

By completing this project, you’ll learn how to:

- Write **semantic and accessible HTML** structure.
- Use **modern CSS** for clean and responsive layouts.
- Apply **typography and spacing** effectively.
- Optimize a static webpage for readability and presentation.
